Final Project

This is my page for my final project for my CSC 106 class. My final project was done by my partner, Nate

Chery and I, where we decided to make a game that was similar to the game of air hockey. In our program we

had a user player and a computer player. The user could use the up and down arrows on the keyboard to move

the user piece. The puck would bounce back and forth between both sides when hit, and when a puck gets past

a player, the other player would receive a point. First player to reach 7 points wins the game. Below is the

link to the game as well as a screenshot of the game itself.
